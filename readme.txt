== Description ==

WP Mobile Redirect is simple user friendly plugin, which allow admin to add link of their mobile version site and redirect mobile users to mobile website version, 
WPMR not only redirect iPhone, Android or Blackberry mobile users, this plugin detect all type of mobile / mobile browsers and redirect all mobile users to that version.
